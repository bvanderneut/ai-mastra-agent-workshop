# Prerequisites Guide

## Introduction

Welcome to the AI Mastra Agent Workshop! This guide contains all the essential prerequisite steps you need to complete **before** attending the workshop. Completing these prerequisites will ensure you can fully participate in the hands-on exercises.

- Basic programming knowledge
- Laptop with installation permissions
- [NodeJS v20 or higher](https://nodejs.org/en/download)
- [GitHub account](https://github.com/)
- [Vercel account](https://vercel.com/)
- [Browser Use account](https://cloud.browser-use.com)

---

## Step 1: Install Required Software

### Node.js Installation

1. Visit [https://nodejs.org/en/download](https://nodejs.org/en/download)
2. Download and install Node.js v20 or higher
3. Follow the installation instructions from the download page
4. Verify installation by running:

```bash
node --version
```

### Git Installation

Ensure Git is installed on your system:

```bash
git --version
```

If not installed, download from [https://git-scm.com/downloads](https://git-scm.com/downloads)

## Step 2: Create Required Accounts

### GitHub Account

1. Create a GitHub account at [https://github.com/](https://github.com/)

### Vercel Account

1. Create a Vercel account at [https://vercel.com/](https://vercel.com/)
2. Sign up using your GitHub account

### Browser Use Account

1. Create a Browser Use account at [https://cloud.browser-use.com](https://cloud.browser-use.com)
2. Verify your email address
3. This service will be used for web automation in our AI agents
